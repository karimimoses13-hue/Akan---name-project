# Akan name Generator
 Akan name Generator project 

 
#Project Description

The Akan Name Generator is a simple  application that helps users discover their Akan name based on their date of birth and gender.

. This project makes it easy and fun to find out your Akan name using basic web technologies.
echnologies Used

#This project was built using:

HTML – for the structure of the web page

CSS – for styling and layout

JavaScript – for logic, calculations, and user interaction


#How the Application Works

The user enters their day, month, and year of birth.

The user selects their gender (male or female).

When the “Get Akan Name” button is clicked:

JavaScript calculates the day of the week the user was born.

The program assigns an Akan name based on the day and gender.

The result is displayed on the page.
